# Smoke checklist перед релизом

**Краткий ручной чек-лист.** Пройти перед выкладкой.

**Автотесты:** `npm run test:e2e` — см. docs/AUTOTEST_PLAN.md.

**Local-safe smoke** (обязательны локально): auth-smoke, create-order-basic (без submit), edit-order-search-smoke (поиск по 78883339999).
**Integration** (staging/preprod): полный create submit, edit-order-extras (допы), compose, gifts, cancel, save-reopen.

---

## Create заказа

- [ ] Рассчитать теплицу, оформить заказ.
- [ ] Заказ появился в Supabase.
- [ ] Заказ виден в боте (тг бот заказы теплицы).

---

## Edit даты

- [ ] Открыть заказ по телефону.
- [ ] Изменить дату доставки → Save.
- [ ] Дата обновилась в Supabase и в боте.

---

## Edit состава

- [ ] Добавить/удалить позицию в составе.
- [ ] Save.
- [ ] Состав обновился. commercial_offer пересобрался.

---

## Edit допов

- [ ] Изменить платные допы (форточка, полив и т.д.).
- [ ] Save.
- [ ] Допы в long КП. Не пропали.

---

## Gift tier up/down

- [ ] Заказ 55k+: 2 слота подарков.
- [ ] Уменьшить сумму до 35k → 1 слот. Toast о смене tier.
- [ ] Увеличить до 75k → 3 слота.

---

## Cancel

- [ ] Отменить заказ.
- [ ] Кнопка «Редактировать» скрыта/неактивна.
- [ ] Попытка save cancelled — блокируется.

---

## Save / Reopen

- [ ] Edit заказа с подарками → Save.
- [ ] Закрыть модалку → открыть по телефону.
- [ ] Подарки восстановились в селектах.

---

## Синхронизация (если трогали)

- [ ] Create → Supabase.
- [ ] Edit → Supabase.
- [ ] Бот показывает актуальные данные.
