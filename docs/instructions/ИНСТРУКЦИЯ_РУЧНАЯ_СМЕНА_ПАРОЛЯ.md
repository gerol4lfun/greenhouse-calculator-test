# Ручная смена пароля и логина (пока админка отключена)

**Когда использовать:** админ-панель в калькуляторе временно отключена. Смена пароля/логина — через Supabase.

---

## Supabase → SQL Editor

### Смена пароля

```sql
UPDATE users 
SET password = 'новый_пароль',
    password_version = COALESCE(password_version, 0) + 1,
    updated_at = NOW()
WHERE login = 'admin';
```

После смены пользователь будет разлогинен при следующей проверке сессии (до 30 сек).

### Смена логина

```sql
UPDATE users 
SET login = 'новый_логин',
    updated_at = NOW()
WHERE login = 'старый_логин';
```

### Деактивация пользователя

```sql
UPDATE users SET is_active = false, updated_at = NOW() WHERE login = 'логин';
```

### Добавление пользователя

```sql
INSERT INTO users (login, password, password_version, is_active, created_at, updated_at)
VALUES ('логин', 'пароль', 1, true, NOW(), NOW());
```

---

## Важно

- Пароли хранятся в открытом виде (plain text). Не использовать чувствительные пароли.
